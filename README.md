# MARS MIPS Assembly Parser 

A robust Python script designed to extract, parse, and structure MIPS assembly memory dumps (Text Segment) generated by the MARS simulator. The extracted data is automatically exported into a clean Excel spreadsheet for further analysis and reporting.

##  Features
- **Regex-Powered Parsing:** Efficiently extracts exact memory addresses, machine code, and original source code.
- **Data Structuring:** Utilizes `pandas` to transform raw text logs into structured DataFrames.
- **Automated Excel Export:** Generates ready-to-use `.xlsx` files without messy formatting.
- **Error Handling:** Gracefully handles missing files, empty logs, and bad formatting.

##  Prerequisites
Ensure you have Python 3.8+ installed. You will also need the following libraries:
- `pandas`
- `openpyxl` (for Excel export)

##  Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/mars-mips-parser.git](https://github.com/mohamed2612001/mips-assembly-extractor.git)
   cd mars-mips-parser```
   
Install dependencies:

```bash
pip install -r requirements.txt
```
Prepare your data:

Export your Text Segment dump from MARS simulator as a .txt file.

Update the file_path variable inside the script to point to your .txt file.

Run the script:

```Bash
python mars_parser.py
```
The script will generate a parsed_data.xlsx file in your specified directory.
